MedSpaNER is a medical text annotation tool for the Spanish language, which is based in hybrid methods (neural networks, rules and a lexicon).

It has been developed using texts about clinical trials. However, it can be applied to other medical text types, and it has been tested with anonymised clinical cases.

How to cite

If you use this tool, you can cite it in the following way:

Leonardo Campillos-Llanos, Ana Valverde-Mateos, Adrián Capllonch-Carrión. Hybrid natural language processing tool for semantic annotation of medical texts in Spanish (Under review)

Evaluation

The trained models have been evaluated twice:

  • Internal evaluation: models were evaluated with a held-out test of 120 texts.

    The RoBERTa models used for the tool obtained the following results (average of 5 experimental rounds):

    • UMLS entities: F1 = 0.886 (±0.002)

    • Medical drug information: F1 = 0.872 (±0.004)

    • Temporal expressions: F1 = 0.900 (±0.001)

    • Miscellaneous medical entities: F1 = 0.677 (±0.003)

    • Negation/speculation: F1 = 0.859 (±0.006)

    • Attributes (temporality/experiencer): F1 = 0.856 (±0.006)

  • External evaluation: 8 health professionals assessed the annotation of 200 new texts.

    The models obtained the following average results:

    • 100 texts about clinical trials: F1 = 0.858 (±0.032)

    • 100 anonymized cases clinical: F1 = 0.910 (±0.019)
